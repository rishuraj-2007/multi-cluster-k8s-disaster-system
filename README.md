# 🚀 Multi-Cluster Kubernetes Disaster System

## 📌 Project Overview
This project demonstrates a production-level Multi-Cluster Kubernetes architecture with disaster recovery and failover system.

## ⚙️ Features
- 🌍 Multi Cluster Setup (Primary + Backup)
- 🔁 Automatic Failover System
- ⚖️ Load Balancing Concept
- 💻 Frontend + Backend Application
- ☸️ Kubernetes Ready Structure

## 🧰 Tech Stack
- Kubernetes
- Docker (future step)
- Python (Flask backend)
- HTML (frontend)
- GitHub

## 📂 Structure
- cluster-a-primary
- cluster-b-backup
- apps (frontend + backend)
- k8s-manifests
- failover-system
- load-balancer

## 💥 Goal
System never goes down — if one cluster fails, another takes over automatically.
